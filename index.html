SOBRE A PAGINA

Pagina interativa feita para ser engraçada e treinar programação web,
utiliza-se html, css e um pouco de javascript. Durante o desenvolvimento
encontrei dificuldades como:

  • O autoplay sendo bloqueado pelo navegador;
  • Imagem de fundo cortada e na frente dos elementos;
  • Cores da pagina muito vibrantes dificultando a leitura;
  • Tabela sem bordas com visual estranho;

Para o autoplay adicionei um pequeno script que quando o usuario
clica na tela, o audio toca, não é a melhor experiencia porem
o navegador não permite audio ser tocado automaticamente

Agora a imagem de fundo realmente causou mais dor de cabeça,
tive que fazer uma div de fundo e usar o css com "::before"
para aplicar a imagem e as mudaças de cor sem afetar o resto.
mas mesmo assim os outros elementos ficavam atraz da imagem
então tive que ajustar o z-index para corrigir, provavelmente
não foi a melhor abordagem, mas o resultado foi como o esperado.

As cores e tabelas foram bem simples de resolver, apenas alterei o css
para mudar as cores e abilitar as linhas na tabela corretamente.

Sobre o formulario ele ainda não apresenta função real pois não
aprendemos sobre javascript ainda.
